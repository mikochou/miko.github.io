368.Largest Divisible Subset

210.Course Schedule II

使用一个visited数组，当visited[i]值为0，说明还没判断这个点；当visited[i]值为1，说明当前的循环正在判断这个点；当visited[i]值为2，说明已经判断过这个点，含义是从这个点往后的所有路径都没有环，认为这个点是安全的。
那么，我们对每个点出发都做这个判断，检查这个点出发的所有路径上是否有环，如果判断过程中找到了当前的正在判断的路径，说明有环；找到了已经判断正常的点，说明往后都不可能存在环，所以认为当前的节点也是安全的。

547.Friend Circles

475.Heaters

radius存储最大半径，对于每个房子，要不就是前面的heater供暖，要不就是后面的heater，所以只需要找出这两个之间的最小值和radius比较，特殊情况是前面或者后面没有heater。

130.Surrounded Regions

找出边界的O，对他们进行bfs查找找出相邻的O，转化为D，然后把剩下的O改为X，D改成O。好像可以用并查集。

1008.Construct Binary Search Tree from Preorder Traversal

返回与给定先序遍历相匹配的二叉搜索树的根结点，二叉搜索树即右子树>根节点>左子树，就写呗

首先用字典保存每个元素最靠右的index，然后遍历S，如果右指针和最靠右的指针相同，那么就是一个section的结束，用左指针标记section的开始。
